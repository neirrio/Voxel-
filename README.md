## ✨ Описание

**Voxel** — это современное Android-приложение для прослушивания музыки, которое объединяет в себе элегантный дизайн и мощные возможности. Приложение использует несколько глобальных источников музыки, предоставляя доступ к миллионам треков со всего мира.

С Voxel вы можете:
- 🔍 Искать музыку из iTunes, Deezer и JioSaavn в одном месте
- 💾 Сохранять любимые треки и создавать плейлисты
- 🎧 Наслаждаться качественным воспроизведением с ExoPlayer
- 📱 Использовать современный Material Design 3 интерфейс
- 🎵 Слушать музыку в фоне с уведомлением плеера

## 🎵 Источники музыки

Приложение использует несколько бесплатных API для поиска музыки:

1. **iTunes Search API** — Официальный API Apple для поиска музыки
2. **Deezer API** — API популярного музыкального сервиса
3. **JioSaavn API** — API индийского музыкального сервиса

Все API работают без ключей и не требуют дополнительной настройки.

## 🔐 Разрешения

Приложение требует следующие разрешения:
- `INTERNET` — Для загрузки музыки и поиска
- `READ_EXTERNAL_STORAGE` — Для доступа к локальной музыке
- `WRITE_EXTERNAL_STORAGE` — Для сохранения музыки
- `FOREGROUND_SERVICE` — Для работы плеера в фоне
- `POST_NOTIFICATIONS` — Для показа уведомлений плеера
